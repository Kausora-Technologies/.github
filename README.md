# Kausora Technologies — .github

Org-wide community health defaults for Kausora Technologies.

## What's Included

| File | Purpose |
| --- | --- |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Community standards and enforcement |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Contribution workflow and conventions |
| [`SECURITY.md`](SECURITY.md) | Responsible disclosure policy |

## How GitHub Applies These

Files in this repository serve as defaults for all Kausora Technologies repositories that do not supply their own. If a repository contains its own copy of any of these files, that copy takes precedence over the one here.

## Proposing Changes

Open a pull request against `main`. Note that changes to files in this repository can affect every Kausora Technologies repository that has not overridden that file — review carefully before merging.
