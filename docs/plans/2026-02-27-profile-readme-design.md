# profile/README.md Design

**Date:** 2026-02-27
**Status:** Approved

## Purpose

Create `profile/README.md` — GitHub renders this as the public-facing org profile page at github.com/KausoraTechnologies.

## Audience

Public visitors: potential clients, collaborators, and developers discovering Kausora on GitHub.

## Approach

Option A — Paragraph intro + capability bullet list + contact links.

## Structure

1. **Heading** — `# Kausora Technologies`
2. **Intro paragraph** — 2–3 sentences: AI, robotics, and automation company; Abu Dhabi and Sheffield; helps organisations scale through intelligent software and industrial systems
3. **What We Build** — bullet list:
   - AI & agentic workflow automation
   - Industrial robotics & autonomous systems
   - Predictive analytics & data intelligence
   - Digital twin & simulation
   - Cloud infrastructure
   - Custom software development
   - Odoo business management (ERP/CRM)
4. **Get in Touch** — website (www.kausora.com) + email (contact@kausora.com)

## Tone

Professional and concise. No badges, no marketing metrics, no emoji. Plain heading only.
